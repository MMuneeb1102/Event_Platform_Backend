## Tech Stack

- **Frontend**: React, Tailwind CSS
- **Backend**: NestJS
- **Database**: Firebase Firestore
- **Auth**: JWT


## Backend Setup (NestJS + Firebase)

### 1. Clone the repo

```bash
git clone https://github.com/MMuneeb1102/Event_Platform_Backend.git
cd event-management-platform/backend

### 2. Npm install

npm install


### 3. Create .env file

MY_SECRET_KEY=your_jwt_secret
PORT=3000


### 4. get service file from Firebase and rename it "firebase-secret.json"

place it into /firebase directory

npm run start:dev


### 5. Run Project

npm run start:dev
